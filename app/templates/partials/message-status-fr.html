Types d’états des messages

GC Notification montre les types d’états de livraison des messages suivants :

Envoi en cours

Cet état signifie que GC Notification a envoyé le message au fournisseur de messagerie ou de téléphonie cellulaire. Si le message texte a été envoyé à l’international, il arrive que certains pays ne fournissent pas d’autres renseignements sur la livraison.

Livraison réussie

Cet état signifie que le message a élé livré avec succès. GC Notification ne vous informera pas si un utilisateur a ouvert ou lu un message.

Échec

Si un message n’arrive pas à être livré, vous pouvez réessayer après 72 heures ou bien essayer de joindre l’utilisateur par un autre moyen. Un message peut échouer pour plusieurs raisons :

  • L’adresse courriel ou le numéro de téléphone n’existe pas

    Dans un tel cas, le fournisseur n’a pas pu livrer le message puisque l’adresse courriel ou le numéro de téléphone était erroné. Vous devriez supprimer cette adresse courriel de votre base de données. Les messages texte envoyés aux numéros qui n’existent pas pourraient tout de même être facturés.

  • Le destinataire n’accepte pas de messages à l’heure actuelle

    Dans un tel cas, le fournisseur n’a pas réussi à livrer le message après avoir essayé pendant 72 heures. Cela peut se produire si la boîte de réception du destinatiare est pleine ou si le téléphone du destinataire est éteint. Vous pouvez essayer d’envoyer le message de nouveau. Les messages texte qui ont été envoyés aux numéros qui n’acceptent pas de messages pourraient tout de même être facturés.

  • Échec technique

    Dans un tel cas, un problème de connexion a eu lieu entre Notification et le fournisseur de messagerie ou de téléphonie cellulaire. Vous devrez essayer d’envoyer vos messages de nouveau. Les messages texte qui sont affectés par une défaillance technique ne vous seront pas facturés.
