The badge component is used to highlight new or important information. It can be used with or without an icon. When the label is meaningful for the understanding of the content it relates to, you should find a way to include the badge text in the accessible name.

Moderate use of this component is key to its success, if there are too many badges, they become overwhelming.

Techniques for making badge text accessible:

  • Don't use a badge if you can use a more descriptive label instead
  • Include the badge text in the accessible name of the related element
  • Pass an id to the badge, and reference it through aria-describedby on the related element.
