Message status

When you send a message using Notification, it moves through different states.

Notification’s real-time dashboard lets you check the status of any message, to see when it was delivered. You can also use our API to check the status of your messages.

For security, this information is only available for 7 days after a message has been sent. You can download a report, including a list of sent messages, for your own records.

Types of message status

These are the types of message status you’ll see when you’re signed in to Notification.

If you’re using our API, some of the statuses you’ll see will be different. Read our documentation for a detailed list of API message statuses.
